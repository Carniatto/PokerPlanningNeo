# Contributing to Poker Planning Neo

Welcome! If you are a developer (human or AI) joining this project, please start by reviewing the following documentation to ensure we maintain our product vision and code quality.

## 📚 Required Reading

Before writing any code, please read:

1.  **[Product Vision](docs/product_vision.md):** Understand the "Neo" philosophy (Modern, Frictionless, Premium).
2.  **[Best Practices](docs/best_practices.md):** Follow our architectural guidelines (Signals, Service-based state, etc.).
3.  **[Glossary](docs/glossary.md):** Use the correct terminology (Host, Round, Reveal).
4.  **[Requirements](docs/requirements.md):** Check the current status and planned features.

## 🚀 Getting Started

1.  **Install Dependencies:** `npm install`
2.  **Run Emulators:** `npm run emulators`
3.  **Start App:** `npm start`

## 🤖 For AI Agents

-   **Always** check `docs/best_practices.md` before refactoring or creating new components.
-   **Update** `docs/requirements.md` when you complete a feature.
-   **Maintain** `task.md` (if available) to track your progress.
